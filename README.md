# Object-Oriented Programming in Java – Advanced Design Patterns & Software Engineering

This repository contains a collection of academic implementations developed as part of an advanced Object-Oriented Programming course in Java IDE.

The course focuses on deep understanding and practical application of object-oriented design principles, software architecture patterns, and aspect-oriented programming paradigms.

## 🧠 Core Topics Covered

The implementations in this repository demonstrate the use of the following concepts:

### 🧩 Object-Oriented Design Patterns
- Interfaces and abstraction design
- Visitor Pattern
- Decorator Pattern
- Composite Pattern
- Observer Pattern
- Template Method Pattern

### 🏗 Software Architecture Concepts
- Dependency Injection (DI)
- Framework-based design principles
- Modular and extensible system design

### ⚙️ Advanced Programming Paradigms
- Aspect-Oriented Programming (AOP)
- AspectJ framework usage
- Separation of cross-cutting concerns

## 💡 Learning Objectives

The main objectives of this course were:

- Understanding advanced object-oriented design principles
- Applying classical design patterns in real software scenarios
- Designing scalable and maintainable software systems
- Learning modular architecture using dependency injection
- Exploring aspect-oriented programming for cross-cutting concerns

## 🛠 Technologies Used

- Java
- Java IDE (IntelliJ / Eclipse)
- AspectJ Framework
- Object-Oriented Programming (OOP)
- Design Patterns (GoF)
- Dependency Injection (DI)
- Aspect-Oriented Programming (AOP)

## 🎯 Skills Demonstrated

This repository demonstrates:

- Architectural thinking and system decomposition
